## 🎨 Paleta de Cores por Seção

### Hero Section
```
Fundo: #0E0E0E (preto) + gradientes primary/10 animados
Título: #FEFEFE (branco) + #ECC80B (amarelo gradiente)
Badge: primary/10 bg + primary border + primary text
Botões: #ECC80B (primário) + border primary (secundário)
Stats: #ECC80B (números) + white (labels)
```

### Benefícios Section
```
Fundo: #0E0E0E (preto)
Cards: white/5 bg + white/10 border
Ícones: primary/10 bg → primary bg (hover)
Gradientes hover:
  - Card 1: from-primary/20 to-primary/5
  - Card 2: from-blue-500/20 to-blue-500/5
  - Card 3: from-purple-500/20 to-purple-500/5
  - Card 4: from-green-500/20 to-green-500/5
  - Card 5: from-orange-500/20 to-orange-500/5
  - Card 6: from-pink-500/20 to-pink-500/5
```

### Processo Section
```
Fundo: white/5 (cinza muito escuro)
Cards: white/5 bg + white/10 border
Números: primary bg + primary border
Linha: from-primary/50 to-transparent (gradiente)
```

### Depoimentos Section
```
Fundo: #0E0E0E (preto)
Cards: white/5 bg + white/10 border → primary/30 (hover)
Aspas: #ECC80B (primary)
Avatares: border-2 border-primary/30
```

### CTA Final Section
```
Fundo: gradient from-primary/10 via-primary/5 to-primary/10
Título: #FEFEFE + #ECC80B (palavra-chave)
Botão: #ECC80B bg + primary-foreground text
Trust badges: primary icons + white/50 text
```

## 📏 Dimensões e Espaçamentos

### Seções
```
Hero: min-h-screen (100vh)
Benefícios: py-32 (128px vertical)
Processo: py-32 (128px vertical)
Depoimentos: py-32 (128px vertical)
CTA Final: py-32 (128px vertical)
```

### Cards
```
Padding: p-8 (32px)
Border Radius: rounded-2xl (16px)
Gap entre cards: gap-8 (32px)
```

### Tipografia
```
Hero título: text-5xl (48px) → text-7xl (72px) → text-8xl (96px)
Seção título: text-4xl (36px) → text-6xl (60px)
Card título: text-2xl (24px)
Badge: text-sm (14px) uppercase tracking-wider
Corpo: text-lg (18px) ou text-xl (20px)
```

### Grid
```
Benefícios: 
  - Mobile: 1 coluna
  - Tablet: 2 colunas
  - Desktop: 3 colunas

Depoimentos:
  - Mobile: 1 coluna
  - Tablet/Desktop: 3 colunas

Stats Hero:
  - Mobile: 2 colunas (2x2)
  - Tablet/Desktop: 4 colunas (1x4)
```

## 🎬 Animações por Seção

### Hero Section
```
Badge:
  - Pulse indicator: animate-ping infinite
  - Entrada: opacity 0→1, y 20→0, duration 0.6s

Título:
  - Entrada: opacity 0→1, y 30→0, duration 0.8s, delay 0.2s

Subtítulo:
  - Entrada: opacity 0→1, y 30→0, duration 0.8s, delay 0.4s

CTAs:
  - Entrada: opacity 0→1, y 30→0, duration 0.8s, delay 0.6s

Stats:
  - Entrada: opacity 0→1, y 40→0, duration 0.8s, delay 0.8s

Parallax:
  - useScroll + useTransform
  - heroY: 0 → 300
  - heroOpacity: 1 → 0
```

### Benefícios Section
```
Header:
  - Entrada: opacity 0→1, y 30→0, duration 0.8s

Cards (cada um):
  - Entrada: opacity 0→1, y 30→0, duration 0.6s
  - Delay: index * 0.1 (escalonado)
  - Hover: gradiente opacity 0→1, duration 500ms
  - Ícone hover: scale 1→1.25, bg change
```

### Processo Section
```
Header:
  - Entrada: opacity 0→1, y 30→0, duration 0.8s

Timeline (cada etapa):
  - Entrada: opacity 0→1, x -30→0, duration 0.6s
  - Delay: index * 0.2 (escalonado)
  
Linha vertical:
  - Sempre visível
  - Gradiente: from-primary/50 to-transparent
```

### Depoimentos Section
```
Header:
  - Entrada: opacity 0→1, y 30→0, duration 0.8s

Cards (cada um):
  - Entrada: opacity 0→1, y 30→0, duration 0.6s
  - Delay: index * 0.2 (escalonado)
  - Hover: border white/10 → primary/30, duration 300ms
```

### CTA Final Section
```
Todo conteúdo:
  - Entrada: opacity 0→1, y 30→0, duration 0.8s
  
Trust badges:
  - Entrada: opacity 0→1, duration 1s, delay 0.5s

Botão:
  - Hover: scale 1.05
  - Ícone: translateX 0→1 no hover
```

## 📱 Breakpoints Responsivos

### Mobile (< 640px)
```
Hero:
  - Título: text-5xl
  - CTAs: flex-col (empilhados)
  - Stats: grid-cols-2 (2x2)
  
Benefícios:
  - Grid: 1 coluna
  
Processo:
  - Timeline: full width
  - Números: menores
  
Depoimentos:
  - Grid: 1 coluna
  - Cards: full width
```

### Tablet (640px - 1024px)
```
Hero:
  - Título: text-7xl
  - CTAs: flex-row
  - Stats: grid-cols-4 (1x4)
  
Benefícios:
  - Grid: 2 colunas
  
Processo:
  - Timeline: full width
  
Depoimentos:
  - Grid: 3 colunas (se couber)
  - ou 2 colunas + 1 abaixo
```

### Desktop (> 1024px)
```
Hero:
  - Título: text-8xl
  - Layout: max-w-5xl centered
  
Benefícios:
  - Grid: 3 colunas (2 linhas)
  
Processo:
  - Timeline: max-w-5xl centered
  - Números: grandes e visíveis
  
Depoimentos:
  - Grid: 3 colunas (1 linha)
  - max-w-6xl centered
```

## 🎯 Hierarquia Visual

### Importância por Peso Visual

```
█████████████████████ Hero Título (Maior)
████████████████ Seções Títulos
██████████ CTAs Principais
████████ Stats Números
██████ Card Títulos
████ Textos Descritivos
██ Labels e Badges
```

### Contraste de Cores

```
Alto Contraste (Atrai atenção):
- Amarelo (#ECC80B) sobre preto
- Branco sobre preto
- Botões primários

Médio Contraste (Conteúdo):
- white/80 sobre preto
- white/70 sobre preto

Baixo Contraste (Secundário):
- white/50 sobre preto
- white/10 borders
```
