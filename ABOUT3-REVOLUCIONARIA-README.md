```typescript

P3: opacity 0→1, y 50→0, delay 0.6s

// Badge final
opacity 0→1, scale 0.8→1, delay 0.8s
```

### DNA Sequenciador
```typescript
// Emojis
whileHover: scale 1.2
onClick: setActiveDNA(index)

// Card
key={activeDNA} // remonta ao trocar
initial: opacity 0, y 30
animate: opacity 1, y 0
```

```typescript
// Estado
```
